使用方法
1. 上传 PDF。
2. 单击普通正文，系统会尽量抓取整段。
3. 如需精确控制范围，直接拖拽框选整段、表格行或多段文字。
4. 右侧 Index 可收起，收起后 PDF 操作区会自动变宽。
5. Word 清单会按照“序号、招股书披露内容、需提供文件、提供情况、公司回复/备注”的表格格式导出。

本机翻译服务启动后，系统会把圈选段落发到本机 127.0.0.1 翻译接口，中文结果会填入右侧“中文翻译”栏，导出前仍可人工修改。
